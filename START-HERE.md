## 🚀 Quick Start - Export in 3 Steps

### Step 1: Build Your Website
```bash
npm run build
```

### Step 2: Deploy (Choose One)

**Option A: Netlify (Easiest)**
1. Go to [netlify.com](https://netlify.com)
2. Drag your `dist` folder onto the page
3. Done! Your site is live in 60 seconds

**Option B: Vercel**
```bash
npm i -g vercel
vercel
```

**Option C: Traditional Hosting**
- Upload contents of `dist` folder via FTP/cPanel

### Step 3: Configure Environment Variables

Set these in your hosting provider:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
```

**That's it! You're live! 🎊**

### Common Issues

**Build fails?**
```bash
rm -rf node_modules dist
npm install
npm run build
```

**404 errors after deployment?**
- Ensure `.htaccess` or `_redirects` file is present
- Check hosting provider's SPA/React Router settings

**Images don't load?**
- Set Supabase environment variables
- Check bucket permissions (must be public)
