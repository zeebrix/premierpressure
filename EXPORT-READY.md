## 🆘 Quick Troubleshooting

### Build Fails?
```bash
rm -rf node_modules dist
npm install
npm run build
```

### 404 Errors on Pages?
- Check redirect configuration (`.htaccess` or `_redirects`)

### Images Don't Load?
- Verify Supabase environment variables
- Check bucket permissions (must be public)

### Forms Don't Work?
- Ensure Supabase Edge Function deployed
- Verify environment variables set

**For detailed solutions**: See [FINAL-EXPORT-GUIDE.md](./FINAL-EXPORT-GUIDE.md) - Troubleshooting section
